# Область проверки SKU Коментарии
1 Проверка прилегающей территории и входной группы.
1.1 Проверка прилегающей территории на санитарное состояние: грязно (листва, окурки, мусор, снег). Не ведутся ремонтные работы (ямы, разбитый асфальт). В зимний период: не очищаются от снега и не посыпаются реагентами или песком дорожки и подходы к магазину, крыльцо, ступеньки. Урны у входа полные мусора, контейнеры для отходов грязные, переполненные баки мусора, вывозятся не регулярно {{ checklistmk.p1_1|default_if_none:"" }} {{ checklistmk.p1_1comment|default_if_none:"" }}
1.2 Санитарное состояние входной группы (грязь, внешние стены и окна – грязные, разрисованные, сломана обшивка, отсутствует закрепленный коврик, отсутствуют ручки на входной двери, не работает доводчик на входной двери, не работает тепловая завеса (заявка отсутствует, истек срок устранения проблемы). Проверка наличия посторонних объявлений на входной двери магазина. Наличие посторонних объявлений, рекламы. {{ checklistmk.p1_2|default_if_none:"" }} {{ checklistmk.p1_2comment|default_if_none:"" }}
1.3 Наличие значка о запрете курения на входной двери магазина. Отсутствие курящих на эстакаде магазина. Курение в магазине запрещено! {{ checklistmk.p1_3|default_if_none:"" }} {{ checklistmk.p1_3comment|default_if_none:"" }}
